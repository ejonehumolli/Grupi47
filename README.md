# Grupi47
Description
This is a simple C++ travel planner program that allows multiple users to plan their trips.
Each person can choose several destinations and specify the number of days, transport method, and travel priority.

The program is designed using basic C++ concepts and is suitable for beginner-level projects.

Features
*Supports up to 4 people
*Each person can choose up to 5 destinations
*Destination selection using initials (e.g., N for New York)
*Choice of transport mode (Flight, Train, Bus)
*Choice of priority (High, Medium, Low)
*Displays a complete itinerary for all users

Concepts Used
*Structures (struct)
*Arrays
*Functions
*for loops
*if / else
*switch statements
*Input and output (cin, cout)

Notes
Destination initials are case-insensitive.
The program focuses on clarity and educational value.
